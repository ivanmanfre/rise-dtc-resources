How this is computed

Deterministic arithmetic, no averages or industry studies. loading…

Assumptions:

  • Returned orders: revenue is refunded and the unit is restocked (COGS recovered), but you still eat outbound shipping, processing fees, and acquisition.
  • Return shipping and handling are not modeled, so the real number is worse than this one.
  • Payment processing uses 2.9% + 30¢, the published Shopify Payments / Stripe standard card rate. Every other default is a labeled example.
  • Repeat lifetime orders use a geometric approximation, 1 / (1 - repeat rate). It is an estimate, not a cohort model.
